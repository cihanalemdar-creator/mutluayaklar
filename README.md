# Mutlu Ayaklar — Hızlı Statik Site

Bu repo, Mutlu Ayaklar için hazırlanmış hızlı, mobil uyumlu ve SEO odaklı statik web sitesi taslağıdır.

## İçerik stratejisi

Site üç ana işi tek çatı altında anlatır:

1. Podolojik ayak bakımı
2. Podo-kozmetik ürünler
3. Podologlara özel eğitimler ve ürün kullanımı

## Teknik yapı

- Framework yok
- Harici font yok
- Harici JavaScript kütüphanesi yok
- Tek HTML + tek CSS + küçük JS
- GitHub Pages / cPanel / Netlify / Vercel üzerinde kolay yayınlanabilir

## Dosyalar

```text
index.html
assets/site.css
assets/site.js
robots.txt
sitemap.xml
404.html
```

## Yayına alma

GitHub Pages için:

1. Repository Settings → Pages
2. Source: `Deploy from a branch`
3. Branch: `main`
4. Folder: `/root`
5. Save

Özel domain bağlanacaksa DNS tarafında ilgili GitHub Pages kayıtları ayrıca girilmelidir.

## Not

Site metinlerinde “tedavi” iddiası yerine “bakım”, “destek”, “yönlendirme” dili tercih edilmiştir. Bunun sebebi sağlık iletişiminde gereksiz regülasyon ve güven riski yaratmamaktır.
